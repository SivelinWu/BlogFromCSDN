# [折腾笔记] 无公网IP如何配置域名映射到NAS

> 原文: https://blog.csdn.net/weixin_42390585/article/details/159432486

> *原创内容，未获授权禁止转载、转发、抄袭。

很多朋友都有自己的nas，但是多数网络运营商不会给公网ip，那么想要在外部访问就会成为一个问题。解决方案有很多种，今天这里讲一种通过Cloudflare Tunnel做内网穿透。
### 1、准备工作
①一个可用的域名
②Cloudflare的账号
### 2、在Cloudflare后台配置连接器
①登录 Cloudflare 控制台（<https://dash.cloudflare.com/>），找到你的域名 `xxxx.com`
②点击Zero Trust
![](/cdn/159432486/9b626fccabdc440eaf71ad4886ca42c8.png)
③点击网络-连接器
![](/cdn/159432486/dc40aeca1b5e491d9feb2d65deb74f03.png)
④点击页面的创建隧道
⑤点击选择隧道类型-Cloudflared
![](/cdn/159432486/c234693b1d8b453b8abb2fd68c473c08.png)
⑥给隧道取一个名，并点击保存
⑦选择操作系统：docker，然后复制下方的token，后面要用
![](/cdn/159432486/cbaf11caf147482fa76af9dcb3c93b1f.png)
⑧填上以下内容，点击完成配置

配置项| 填写值| 说明  
---|---|---  
服务类型| `HTTP`| 对应NAS的 HTTP服务  
主机名| `nas`| 对应你的三级域名`nas.abc.com`  
域名| `abc.com`| 你的主域名，自动补全  
服务 URL| `http://[极空间NAS内网IP]:5055`| 比如`https://192.168.1.10:5055`（极空间 HTTPS 默认端口，必须和你直接访问 NAS 的端口一致）  

![](/cdn/159432486/f8c6bea525d1434ea41286d46923bea5.png)
### 3、进入NAS（本文中以极空间nas为例）安装Cloudflare
①搜索Cloudflare/Cloudflared镜像并下载
![](/cdn/159432486/0227c3198da34cdebbf144e4bab1d8ce.png)
②下载完成后双击安装，装载路径填 /etc/cloudflared
![](/cdn/159432486/8a369a3ab85046d59a9b54565ae1a445.png)
③网络要选host，千万不要选bridge！
![](/cdn/159432486/84d326ec21b54e8b92476bc26c4c2979.png)
④命令这里选择自定义，然后贴上下面的命令：
    
    tunnel --no-autoupdate run --token [前面步骤从Cloudflare复制的token]
![](/cdn/159432486/fd7eef552a57400b951c562cc60c78d8.png)
⑤点击完成，启动容器，等几分钟，启动完成后就可以用<http://nas.abc.com>访问我们的nas了。
![](/cdn/159432486/a35661b716424b6f92a035bb32985708.png)
一切大功告成！
